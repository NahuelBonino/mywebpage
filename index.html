Proyectos personales

Casas en Villa Soriano
Actualmente mantengo una página que desarrollé hace algunos meses. La persona que me pidió esta página es dueña de varias casas en Villa Soriano. Querí mostrar sus casas en un sitio personalizado, con un dominio elegido. Luego de evaluarlo se compró el dominio casasvillasoriano.com.uy y se levanto el sitio web en un servidor de un prestador de servicios de hosting, quierohosting. Esta página la construí con HTML, CSS junto con los lenguajes de programación PHP y JavaScript. Además utilizé un front-end framework llamado Bootstrap.

Una funcionalidad interesante es la de comentarios, donde un cliente puede dar una devolución de su estadía. La funcionalidad de comentar la implementé con el lenguaje JavaScript utilizando la base de datos no relacional de Firebase. Esta herramienta es bastante beneficiosa para este proyecto, aprovechando que es un sitio pequeño y de pocas visitas, ya que es gratuita y fácil de utilizar. Dejo acceso a una página de prueba dentro del sitio para probar los comentarios casasvillasoriano.com.uy/pueba.

GloboDreams
Este es un proyecto que he ido postergando en el tiempo. Se trata de un Juego para Android dividido en niveles donde se tiene que superar uno para avanzar al siguiente. Cada nivel es conformado por uno o varios objetivos que se deben superar, todos los objetivos estan relacionados por explotar globos. Estos niveles a medida que aumentan se encuentran en lugares cada vez mas alejados de la tierra.

Todavía sigo desarrollandolo, utilizo la plataforma Unity para hacerlo, con el legunaje C#.

2Teams
Es un proyecto desarrollado en el entorno de desarrollo Android Studio, con el Lenguaje Java. Se trata de una aplicación que dada una cantidad de no más de diez personas, los divide en dos equipos al azar de iguales cantidades.
